OrderFlow - REST API 주문관리

Spring Boot 3.3.5 · Java 21

Spring Boot JPA/Hibernate QueryDSL 5.0 JWT MySQL Swagger

Spring Boot와 JPA를 활용한 주문 관리 REST API 시스템입니다. JWT 인증, QueryDSL 동적 검색, 주문 취소 시 재고 복구 등 실무 패턴을 구현했습니다.

  • JWT 기반 인증/인가 (Access Token)
  • QueryDSL 동적 검색 (이름, 가격, 상태 필터)
  • 주문 취소 비즈니스 로직 (재고 자동 복구)
  • Swagger API 문서 자동화
  • 개발 기간: 2025.11.10 ~ 12.10

Bookmarket - 온라인 서점

Spring Security · 아임포트 결제

Spring Boot 3.4 Spring Security OAuth2 아임포트 API JPA Thymeleaf MariaDB

Spring Security를 활용한 인증/인가와 아임포트 결제 API를 연동한 온라인 서점입니다. 회원 관리, 상품 CRUD, 주문/결제 전체 흐름을 구현했습니다.

  • Spring Security + BCrypt 인증/인가
  • 아임포트 외부 결제 API 연동
  • ROLE_USER / ROLE_ADMIN 권한 분리
  • Log4j2 + Interceptor 모니터링
  • JPA Entity 연관관계 설계
